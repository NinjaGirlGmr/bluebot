This directory contains the robot's printable 3D part models (`.stl`).

- `Camera deck.stl`
- `Camera Mount.stl`

## Notes

- Open these files in your slicer/CAD tool to inspect orientation and scaling before printing.
- Keep part filenames stable to avoid breaking references in documentation and build notes.
